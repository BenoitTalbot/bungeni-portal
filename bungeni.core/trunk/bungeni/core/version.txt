Versioning
------------------------

  >>> import bungeni.models.domain
  
  >>> import bungeni.core.version
  >>> component.provideAdapter(
  ...    bungeni.core.version.ContextVersioned,
  ...    (bungeni.core.interfaces.IVersionable,),
  ...    bungeni.core.interfaces.IVersioned)  


  >>> question = bungeni.models.domain.Question()

  >>> from zope.interface import classImplements
  >>> classImplements( bungeni.models.domain.Question, bungeni.core.interfaces.IVersionable)


check that there are no versions yet:
  >>> versions =  bungeni.core.interfaces.IVersioned(question)

  >>> len(list(versions.values()))
  0  
  
create a version:  
  >>> versions.create('New version created ...')

  >>> len(list(versions.values()))
  1
